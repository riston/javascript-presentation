Where is Javascript used ?

  • part of web browsers
  • client side scripts
  • interact with the user
  • alter the document that is displyed
  • works "almost" cross-browser and platform

Server side

  • Node.js is a packaged compilation of Google's V8 JavaScript engine
  • ... write same language on client-side and server-side.
  • ... high throughput via non-blocking I/O

Based on Javascript

  • CoffeScript ES5 "strict mode" subset (AKA the "good parts") of JavaScript. That means no with statement, no loose equality checking, etc.
  • TypeScript offers classes, modules, and interfaces to help you build robust components.
  • Asm.js an extraordinarily optimizable, low-level subset of JavaScript.

Languages which compiles to Javascript

  • Dart is a class-based, single inheritance, object-oriented language with C-style syntax.
  • ClojureScript compiler for Clojure that targets JavaScript.

Conclusion

  • Javascript as a language has improved a lot in recent years
  • Hopefully becomes better with ECMAScript 6 implementation
